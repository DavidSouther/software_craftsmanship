*   Students will be able to describe an **object**
    *   An object is a piece of structured data.
    *   An object can be stored in a variable.
    *   An object has **properties**
    *   Properties are like variables, and have values.
*   Students will be able to access properties on built-in Date objects.
*   Students will be able to create and modify built-in Date objects.
*   Students will be able to describe a **class** as a way to bundle data into **objects**
*   Students will be able to describe **methods** as functions of a **class**
*   Students will be able to create a **class** using a **constructor** to **initialize** properties
*   Students will be able to update class data using methods
*   Students will be able to **extend** a class to share functionality.
