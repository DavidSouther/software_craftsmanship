# Python Cheatsheet

1. Python Folder - 
  * On Desktop, a folder called TEALS_Python
1. Open Idle
1. New File
1. Type your program
1. Save As `filename.py` in Desktop/TEALS_Python
1. Open Terminal (Application / Utilities)
1. Type and hit enter:
  1. `cd ~/Desktop/TEALS_Python`
  1. `python filename.py`
